How to Use GitHub Discussions

1

Enable GitHub Discussions

Go to your repository Settings → Features → Enable Discussions

2

Create Discussion Categories

Set up categories: Fundamental Analysis, Technical Analysis, News & Updates, Earnings Reports

3

Link Stocks to Discussions

When adding a stock to stocks.json, include the GitHub Discussion URL in the discussionUrl field

4

Automatic Integration

This page will automatically fetch and display discussions using the GitHub API (no authentication required for public repos)

Note: To enable automatic discussion fetching, update the repository URL in assets/js/discussions.js with your actual GitHub organization and repository name.
